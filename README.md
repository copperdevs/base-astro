# Astro + React Example

## Getting Started

1. clone or whatever
2. `npm install` or `bun install` or your preferered thing
3. edit [`/src/appInfo.ts`](/src/appInfo.ts) to update it to your prefered info
4. `npm run dev` or `bun run dev`
